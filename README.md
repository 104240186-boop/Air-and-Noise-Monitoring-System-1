# Smart Air Quality and Noise Monitoring System

This project presents a portable IoT-based environmental monitoring system designed to measure indoor air quality and environmental noise levels in real time.

The system uses an ESP32 microcontroller as the main processing unit, an MQ135 sensor to estimate CO₂ concentration and evaluate air quality, and an MKE-S06 sound sensor to monitor ambient noise levels. Collected data are processed by the ESP32 and transmitted to the Blynk cloud platform via Wi-Fi, allowing users to remotely monitor environmental conditions through a smartphone application.

Key features of the system include:

* Real-time monitoring of indoor CO₂ concentration.
* Real-time environmental noise measurement.
* Wireless data transmission using Wi-Fi.
* Remote visualization through the Blynk IoT platform.
* Threshold-based warning notifications for poor air quality and excessive noise levels.

This project aims to provide a low-cost and portable solution for smart environmental monitoring in classrooms, offices, factories, hospitals, and smart home applications.
